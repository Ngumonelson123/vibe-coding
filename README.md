# Product Gallery

![Product Gallery](Screenshot%20from%202025-11-10%2020-50-21.png)

A responsive 3-card product showcase with light/dark mode toggle and smooth animations built with vanilla HTML, CSS, and JavaScript.

## Features

- **3-Card Layout**: Product cards with images, titles, descriptions, and prices
- **Theme Toggle**: Light/dark mode with persistent storage
- **Hover Effects**: Card lift, image zoom, and shadow animations
- **Fade-in Animations**: Staggered card entrance animations
- **Responsive Design**: Mobile-friendly grid layout
